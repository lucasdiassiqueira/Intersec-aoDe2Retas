Interseção de Duas Retas no Plano

O Que É Interseção?

Quando duas retas se cruzam em um ponto no plano, dizemos que elas possuem uma interseção. Esse ponto é a solução do sistema formado pelas equações das duas retas. Se elas forem paralelas e distintas, não haverá interseção. Se forem coincidentes, têm infinitas interseções.

Como Calcular?

Você pode resolver um sistema de equações do tipo y = m1x + b1 e y = m2x + b2 para encontrar o ponto de interseção. Igualando as equações, resolvemos para x e substituímos em uma das expressões para encontrar y.

Exemplo

Considere as retas y = 2x + 1 e y = -x + 4. Igualando:
2x + 1 = -x + 4
3x = 3 ⇒ x = 1
Substituindo: y = 2(1) + 1 = 3
Ponto de interseção: (1, 3)
